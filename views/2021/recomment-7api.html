                    分享给大家 7 个超级棒的 API

                    掘金安东尼 2021-06-28 API

                    # 小序

                    众所周知，“前端工程师”又称为“API 调用大师”，API 调用几乎充斥着我们的开发生活。

                    好的 API 赋给我们足够的能量（动态数据），让我们可以一次次有力地打出精彩的组合拳（组装应用）！

                    鲁迅先生曾说过这么一句话：

                    “ API 调的好，下班下的早！”

                    本瓜严谨治学，翻阅典籍，找到原图，石锤他老人家的确说过！！请放心引用！！

                    本篇将分享给大家 7 个超级棒的 API，点赞👍👍👍 收藏⭐⭐⭐，赚到了🤭🤭🤭

                    本篇译自 medium：7 Awesome APIs for All Frontend Developers (opens new window)，6 月 19 号发的，一周不到，还有个热乎劲，赶快来看~

                    # The MovieDB API

                    如果你要做一个电影相关的 app，The MovieDB API (opens new window) 一定不要错过。它能让你很轻松的获取到电影数据、演员剧照、电视节目等等。

                    它的文档也非常详细，对于电影爱好者也是福音~

                    # The NASA APIs

                    NASA API (opens new window) 让你可以访问有关天文学、太空天气、月球、火星等等不同类型的天文数据。

                    天文爱好前端儿得福音~

                    # Frankfurter

                    Frankfurter API (opens new window)，可让你获取货币数据并跟踪可用汇率。

                    # The MealDB API

                    The MealDB API (opens new window)，顾名思义，干饭数据来的。它可以让你获取有关膳食数据、视频，包括膳食成分、图像、类别、视频等等数据。

                    简直不要太强~ 吃货前端开发儿的福音~

                    # Responsive Voice API

                    Responsive Voice API (opens new window)，它不仅允许你将任何文本转换为语音，它还有很多其它功能，例如更改说话的口音、性别、音量等等。

                    它有详细的说明文档！

                    变声怪的福音~自己开发变声器它不香吗？

                    # Count API

                    Count API (opens new window)，使你能够创建数字计数器跟踪记录访问页面的次数。不止于此，如果你想获得某个按钮的点击次数，也是 OK 的！

                    如果你有建立自己的 Blog，需要记录页面访问次数、按钮访问次数，那么建议你尝试调用它！

                    # FreeToGame API

                    FreeToGame API (opens new window)！游戏党前端儿必备！

                    它提供了按类别、平台、标签等过滤游戏数据的能力。你也可以从特定游戏中获取详游戏细信息！

                    # 小结

                    以上 API 都是很好玩、很有用的 API，它为咱前端开发提供了宝贵的动态数据。希望你能用到！

                    如果暂时用不到，可以 mark 标记一下，以供日后再用~
