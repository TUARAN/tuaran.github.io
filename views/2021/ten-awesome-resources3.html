                      掘金安东尼 2021-10-19 前端

                      小知识，大挑战！本文正在参与「程序员必备小知识 (opens new window)」创作活动

                      本文已参与 「掘力星计划」 (opens new window) ，赢取创作大礼包，挑战创作激励金。

                      本篇是 10 个 “哇塞” web 资源系列 第 3 篇！前两篇传送门：

                      • 《推荐 10 个很“哇塞”的Web“资源”给前端工友，收藏等于学会~》 (opens new window)
                      • 《1234 再来一次，继续分享新 10 个“哇塞”的 web 资源，收藏等于学会~》 (opens new window)

                      不得不说，web 世界缤彩纷呈，好资源总叫人爱不释手！开放与接纳成就了如今的万维网世界🌎，一个链接，一段故事🎈~

                      # 1. builtwith (opens new window)

                      https://trends.builtwith.com/ (opens new window) 一个神器的网站，可以告诉你当前市面上 Web 技术的使用数据、趋势等；

                      🌰比如说，你想查一下，当前市面上，还有哪些网站在使用 JQuery，就可以在 https://trends.builtwith.com/websitelist/jQuery (opens new window) 中找到！

                      天呐！竟然还有至少 7 千万+ 个网站在使用着 Jquery！

                      本瓜再查了下正使用 Vue 的站点是 2 百万+ 个，React 是 1 千万+ 个；

                      卧了个槽，Jquery，YYDS ！！

                      # 2. zety (opens new window)

                      免费简历制作，这风格 真职场、ui 真清爽~

                      讲道理，简历形式上做成这样就足够了，简单明了，标准规范，更多就交给实质干料了；

                      # 3. cheatography (opens new window)

                      小抄网站，实现程序猿的各种打小抄梦想！

                      关于技术、商业、教育、家庭、健康、游戏，多达 4000 + 个小抄，真 “好记性不如打小抄”！

                      🌰比如 JavaScript 小抄，果断把这个 PDF 下载到本地收藏了：

                      # 5. canva (opens new window)

                      再来亿个大家都爱看的设计网站，https://www.canva.com/zh_cn/ (opens new window) 收藏就完事了，绝不会让你失望~

                      真滴顶👍 再也不用担心配图设计啥的了~

                      做网站的 SEO，一定不能错过 Alexa！查看网站数据、排名、搜索关键词、竞品网站等信息；

                      # 7. emaildrop (opens new window)

                      通过这个网站，你可以拿到一个 一次性使用 的电子邮件地址✉，如有需要，还可以将其升级为永久邮件地址；

                      # 8. generative.fm (opens new window)

                      本瓜太爱这个网站了~

                      它能产生 永不结束、永不重复 的纯音乐（环境音）；敲代码的时候听再合适不过了！！

                      感觉把这个网站推广给各大咖啡店也十分合适啊，氛围拉满~

                      不要再问 VScode 有什么好的插件了！最受欢迎的全在 vsce (opens new window) 这里了~ 自己去挑吧！
