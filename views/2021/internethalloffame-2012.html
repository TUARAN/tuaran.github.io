      互联网名人堂——“创新者”（2012 至 2019 入选合集）

      掘金安东尼 2021-09-12 程序员

      # 小序

      摸鱼🐟摸到：原来咱互联网也是有名人堂的~😮 撰文记之，与大家分享 ~φ(゜▽゜*)♪

      互联网名人堂由 ISOC（国际互联网协会） 发起，旨在纪念和感谢那些获得了全世界认同的、为全球开放的互联网发展和完善做出了重要贡献的杰出人物。

      奖项又细分为 3 类：互联网开拓者🚀、互联网全球性互联者🌏、互联网创新者💡；

      本篇将带来自 2012 年奖项成立以来，被颁发的所有【创新者奖项】的“真”大佬们的介绍~~

      *由于一些因素，评奖并不是每年都进行，截至目前，举办过评选的年份是：2012 年、2013 年、2014 年、2017 年、2019 年。

      文末附有本瓜关于“名人堂”的两三点看法，好像说了，又好像没说🐶

      如果觉得本篇内容还不错，欢迎点赞/收藏/评论/关注👍

      闲言少叙，正篇开启<(￣︶￣)↗冲

      # 2012

      # 1. Winifred Mitchell Baker

      第 1 届第 1 名是一位杰出的女性，Mozilla 基金和 Mozilla 公司的主席；Mozilla 证明了网络不应该属于一个公司和一个操作系统；

      # 2. Tim Berners-Lee

      隆重介绍这位🎉 蒂姆·伯纳斯·李

      万维网、Web服务器、Web浏览器、HTTP和HTML的发明人！！

      他（英国人）在 12 年伦敦奥运会上发了一条著名的 twitter —— "This is for Everyone"，万维网是给每个人的~~

      # 3. Robert Cailliau

      这位和楼上那位联合创建了万维网，并为苹果 Mac 电脑开发了第一款 Web 浏览器；👏

      # 4. Van Jacobson

      TCP流量控制算法的提出人，该算法使得网络规模可控；

      # 5. Larry Landweber

      阿帕网替代网络 CSNET 的开发人，CSNET 在全球 180 所高校被广泛使用；

      1991年，他（手动）绘制了一份互联网世界分布图；

      1997 年，Internet 已覆盖全球🌏

      # 6. Paul Mockapetris

      开发了互联网分布式及动态域名系统，DNS 🐂🍺

      # 7. Craig Newmark

      Craigslist 网站的创始人；在美国，他的知名度直逼好莱坞大腕。craigslist 网站以邮件列表的形式实现了社区化的交流服务，是全球最大的信息播报平台。

      # 8. Raymond Tomlinson

      阿帕网第一个电子邮件系统的发明人；邮件要用 @ 符号，就是他发明的~👍

      # 9. Linus Torvalds

      这位老哥应该无人不知、无人不晓吧！！😀

      Linux 内核&& Git 的作者！！

      还有句著名的：

      Talk is Cheap,Show me the Code. —— Linus Torvalds

      🐶🐶🐶

      # 10. Philip Zimmermann

      隐私和安全倡导者，电邮加密系统优良保密协议（Pretty Good Privacy）的开发者；

      # 2013

      # 1. arc Andreessen

      网景浏览器创始人；

      # 2. enning Schulzrinne

      会话发起协议（SIP）的主要设计者之一；

      # 3. ohn Perry Barlow

      电子前线基金会创始人之一，是一个国际知名的民权组织，旨在维护互联网上的公民自由、提供法律援助、监督执法机构；

      # 4. ichard Stallman

      通用公共许可协议以及自由软件基金会的创立者；他还是 Emacs、GCC 的必要贡献者，并且是创建 GPL 的基础，可以毫不夸张地说，没有 [Stallman]，开源软件就不会存在；

      # 5. nne-Marie Eklund Lowinder

      瑞典网络安全领域的先驱；

      # 6. aron Swartz

      Reddit联合创始人、RSS 规格合作创造者；

      # 7. rancois Fluckiger

      欧洲核子研究组织计算机院主任、《网络多媒体开发与应用》作者；

      # 8. immy Wales

      维基百科创始人之一；本瓜太爱 wiki了！salute！

      # 9. tephen Kent

      BBN 公司首席科学家、网络安全专家；

      # 2014

      # 1. Eric Allman

      Delivermail 作者，对当今使用的电子邮件传输技术产生了重大影响；

      # 2. Eric Bina

      网景公司联合创始人，担任程序员时开发了 Mosaic 首个版本；

      # 3. Karlheinz Brandenburg

      MP3 音频数据压缩方法作者之一；

      # 4. John Cioffi

      电气工程师，教育家和发明家，他在电信系统理论，特别是编码理论和信息理论方面做出了贡献。Cioffi 被誉为“ DSL之父”，其开创性的研究在使数字用户线路技术实用化方面发挥了重要作用；

      # 5. 钱华林💖

      中国人！！！！！把骄傲刷起来！！！！！

      他是中国互联网重要的开创者之一，是“中国互联网先驱”的“二钱”之一（另一位是钱天白）；

      * 温顿·瑟夫：TCP/IP协议的发明者；

      # 6. Paul Vixie

      他的技术贡献包括域名系统协议设计和过程，实现 DNS 实现的操作稳定性的机制，以及对开源软件原理和方法的重大贡献；

      # 2017

      # 1. Jaap Akkerhuis

      自 1980 年代初以来，Jaap Akkerhuis 一直在荷兰和欧洲的互联网发展中发挥着重要作用；

      # 2. Yvonne Marie Andrés

      Andrés 博士是最早利用 Internet 为教育工作者和学生开发和推进全球电子学习机会的人之一；

      # 3. Alan Emtage

      现代搜索引擎的祖先；

      他构思并实施了第一版 阿奇，一个网页互联网搜索引擎，于在公共场所查找资料的 FTP 档案。它被广泛认为是世界上第一个 互联网搜索引擎；

      # 4. Ed Krol

      Ed Krol 是国家超级计算应用中心的网络管理员；

      # 5. Tracy LaQuey Parker

      互联网的早期传播者之一；

      # 6. Craig Partridge

      Craig Partridge 博士对互联网从普通实验网络到全球通信基础设施的早期技术发展做出了重大贡献；

      除此之外，他还设计了如何使用域名路由电子邮件；领导开发第一台高速（千兆位）路由器的团队；并与 Stephen Kent 博士一起参与了开发高性能 IP 加密设备架构的团队。他还是任播寻址的共同发明者，并为 TCP（传输控制协议）往返时间估计做出了贡献；

      # 2019

      # 1. Jean Armour Polly

      Jean Armor Polly 是最早向公众提供计算机和互联网访问的美国图书馆员之一，改变了公共图书馆服务并重新定义了图书馆员作为数字教育者和互联网倡导者的角色；

      # 2. Jose Soriano

      “公共互联网信息亭”模式的主要架构师和倡导者，该模式使网络访问民主化；这种模式被世界银行宣布为最适合发展中国家的模式；

      # 3. Michael Stanton

      促使巴西政府建立了该国第一个国家研究网络，组织讨论建立一个单一的国家可以访问外部世界的研究网络；

      # 4. Klaas Wierenga

      Klaas Wierenga 发明了 eduroam，为全球 100 多个国家/地区提供服务，帮助增加了大学和研究团体之间的合作；

      # 5. Suguru Yamaguchi

      是网络安全研究的先驱和部署方面的全球领导者；他也是 WIDE（广泛集成分布式环境）项目的创始成员；致力于研究网络安全用于大规模分布式计算环境的系统；

      # 小思

      本瓜看着互联网名人堂和 NBA 名人堂有种类似的感受：这些人好强啊！呀！这个原来是他发明的呀！真强啊！还是地球人嘛？......

      emm......不过就是：咱中国面孔、或者亚洲面孔太少了；

      没办法，由于一些先天不可抗因素，就好比 NBA 中，咱体质上就是比不过人家，而互联网，先天性就是起步晚，按老话讲还是舶来品；

      国内的程序员普遍有 35 岁的悬顶之剑，与国外一对比，有着明显的差异，他们职业生涯更长，其中有成就的程序员一般大都是老头~~

      确实，少有人能在短短数载就能成为行业卓越吧~~

      要思考的是：我们在软件业务上是不是过于浮躁，一个项目往往只有几个月的生命周期，好的情况有一年以上。更多的时候都是追热点吧？是否该倾斜一点重心到技术的基础建设当中？

      其次，本瓜认为，评选名人堂还是很有必要的！这些了不起的人起到了指引作用，比喻为灯塔一点不为过；任何领域，名人效应 都很重要，就像中国篮球需要姚明，中国田径需要刘翔、苏炳添，中国钢琴需要朗朗......

      最后，以上这些互联网名人大部分都是在自己的细分领域做大做强（极个别是全能型能打选手），所以，专注成就伟大。
