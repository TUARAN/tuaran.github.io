是时候废除“网络君主制”啦？认识 Usenet ~

掘金安东尼 2022-04-16 程序员

一起养成写作习惯！这是我参与「掘金日新计划 · 4 月更文挑战」的第6天，点击查看活动详情 (opens new window)。

# 背景了解

刚看到一篇文章：《是时候废除“网络君主制”了》 (opens new window)，大致的意思是目前我们的社区、论坛都很“中心化”，就像是君主制度，社区规则由极少数人制定，社区资源由极少数人掌握，这很不民主。

于是乎，作者提出这样的设想：我们可以选举“社区管理员”，不仅如此，还可以进一步获取“社区资源”（比如数据库等等）；当我们不满意当前的管理时，可以基于社区目前的状态再分割出一个新社区，这有点像 git 分支管理中再拉取一个分支一样。

简单将它理解为：目前的网络社区就像 svn 一样，集中管理，而改进后，应该是 git 那样分布式的；

随后咱又了解到，其实早已经有一种分布式的互联网交流系统了，它便是 —— Usenet

Usenet 什么来路？

它于 1979 年被设计，1980 年发布，全名是：users network ；

这个时间有多早？要知道 Windows XP 过去在中国高普及度，IE 6.0 是许多人接触的第一个浏览器。而在 IE6 之前还有5个大版本，IE 浏览器的第一个版本发布于 1995 年，而 Usenet 比初版的IE浏览器还早了15年。

堪称：古早！

Usenet 可以说是现在论坛的祖师爷，它推广了许多广为人知的概念和术语，如 FAQ、马甲、网络论战、垃圾消息等等。

# 运行机制

Usenet 运行机制分 3 步骤：

  1. 在 ISP 网络提供商那买一台服务器作为 Usenet 专用服务器，得到一个网址；（类似注册 git 账号，不过非免费）
  2. 发帖：在这个网址上发帖；（类似 git push 操作）
  3. 查看帖：从服务器上下载更新其他人的帖；（类似 git pull操作）

它有一个特定的客户端：

Usenet 服务器每天会同其他Usenet 服务器交换帖子，也就是说会保持内容的同步，即无论你的帖子是发到哪一台服务器上，最终全世界的人们都会看到；

这样，Usenet 就实现了分布式～

# 更多Q&A

  • Usenet 今天还在使用吗？

目前 Usenet 比 Internet 上的用户数量要少很多，但是仍有人在使用它；它提供了更加安全的方式来共享信息；并且，还有许多 Usenet 供应商在运作，说明它仍有需求；

  • Usenet 比 BitTorrent 更安全吗？

支持 SSL 和 VPN 的 Usenet ，比 BitTorrent 更安全；

  • Usenet合法吗？

Usenet 完全合法的。Usenet 中的一些新闻组通常都有自己的规则，禁止上传非法内容。

  • Usenet是暗网吗？

它不是暗网，相反，它是一个透明的共享信息的网络。

  • Usenet 和 Internet有什么区别？

Usenet 不是 Internet 。它是一个单独的网络，Usenet 主要用于文件共享。Usenet 中的服务器使用 NNTP 协议来交换数据。而 Internet 是一个通过 TCP/IP 进行通信的互连计算机网络；

小结：我们了解到：其实分布式的社区论坛早就有了，也就是 Usenet，不过没有成为主流的原因可能由于第一：不算方便、第二：要花钱买专属服务器、第三：如今的论坛也是百花齐放，这个不行就选那个；但，非要去设想我们什么时候会回到 Usenet 呢？那大概就是市面上出现了大量论坛、主流论坛突然崩掉、倒闭，或xx太红色了，或者出现侵犯用户隐私丑闻等等诸如此类吧~

从新冠这事来看，现在这个社会：什么都有可能发生，不是吗？
