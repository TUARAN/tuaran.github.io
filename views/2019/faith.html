                  # 信用财富

                  楚人谚曰：“得黄金百，不如得季布一诺”

                  # 信用是无形的资产

                  从大三起在分期乐上为虚荣分期买的第一台phone7起，如今好似已逐步堕入了**“借贷的深渊”**。

                  细想有过借钱去旅游、借钱坐飞机（吃机场五六十一碗的面条）、借钱去借别人钱等荒唐行径，最后就变成了借钱去还借的钱，直到借款的平台的年利率到了35.99%（法定超24%无司法保护，超36%无效），这让人不寒而栗的数字。

                  从计算负债1万，到2万，到3万，不知道会不会在工作以后停止继续增长的负债？但是事总是与愿违，毕业时及毕业后庞大的开销使得借钱的背负感没有一丝减轻，并且还加深了对“应有储蓄”的担忧和恐惧感。

                  然而较为舒适的生活使我挣扎，虚荣的欲望使我迷失。

                  然后到了某一时刻，该发生的总是需要去发生，因为自己清楚的明白自己不是那样的人，可以更克制些，更机灵些，所以我需要且必须开源节流。

                  从心底我是害怕清算财产的，但是不能等到这一切绷不住了，我再去做，现在应该还来的及！

                  我不能自我蚕食信用

                  想一想：2.5号过年，所以估计工资也就还剩10月、11月、12月、1月，共计4个月，估计也就两万多一点，如果想存一万块回家过年？怎么能够？
